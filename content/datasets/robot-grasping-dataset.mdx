A comprehensive dataset for robot grasping research containing RGB-D images, object poses, and grasp annotations.

## Contents

- 10,000 RGB-D images of household objects
- 3D object models and pose annotations
- Grasp success labels and trajectories

## Usage

Download the dataset and extract it to your working directory. See the README included in the archive for detailed usage instructions.
